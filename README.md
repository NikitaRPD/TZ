# Техническое задание (TZ)

## Описание задачи

Доделать микросервис, представляющий собой API с двумя эндпоинтами для управления хранилищем книг.

Цель — реализовать недостающие части сервиса и убедиться, что все тесты проходят.

---

## Требования к реализации

1. **Модели**
   - Создать необходимые модели для работы с данными книг и авторов.
   - Обеспечить связь между книгами и авторами.

2. **Миграции**
   - Сгенерировать и применить миграции для создания таблиц в базе данных.

3. **Эндпоинты**
   - Реализовать описанные ниже эндпоинты:
     - `POST /api/books/` — добавление новой книги.
     - `GET /api/books/` — получение списка всех книг.

4. **Тестирование**
   - Все тесты из `tests/` должны проходить.

---

## Запуск

> Требуется: Docker и Docker Compose.

### 1. Поднять базу данных и сервис:

```bash
docker-compose up -d
```

Сервис доступен на http://localhost:8099, Swagger — http://localhost:8099/docs.

### 2. Создать и применить миграции (после реализации моделей):

```bash
docker-compose run --rm tz-app uv run alembic upgrade head
docker-compose run --rm tz-app uv run alembic revision --autogenerate -m "books"
docker-compose run --rm tz-app uv run alembic upgrade head
```

> После изменения кода пересобрать образ: `docker-compose up --build -d`

### 3. Запустить тесты:

```bash
docker-compose --profile test run --build --rm tz-tests
```

> `--build` обязателен — без него Docker может использовать старый образ и не увидеть новые тесты.

## Функциональные требования

### API Эндпоинты

1. **Добавление новой книги**
   - **Метод:** `POST`
   - **URL:** `/api/books/`
   - **Структура запроса:**
     ```json
     {
       "title": "The Great Gatsby",
       "authors": [
         "123e4567-e89b-12d3-a456-426614174000"
       ],
       "genre": "Novel"
     }
     ```

2. **Получение списка всех книг**
   - **Метод:** `GET`
   - **URL:** `/api/books/`
   - **Структура ответа:**
     ```json
     [
       {
         "id": "e2fcebf1-dfe5-4ddd-9cef-5c25c76afca8",
         "title": "The Great Gatsby",
         "authors": [
           "123e4567-e89b-12d3-a456-426614174000"
         ],
         "genre": "Novel"
       }
     ]
     ```
